Program DesignÔÉÅ

The goal is to create an object-oriented Finite Element Analysis program that

  1. is based on the finite element method

  2. handles arbitrary trusses in 2D

  3. allows for arbitrary loading

  4. can plot its undeformed and deformed shape

The program will need and use the following objects:

  • Mesh Generators
    • Lines and Curves
    • Quadrilateral domains
    • Triangular domains
    • 3D Solid domains
  • Node
  • Element
  • Material
  • System
  • Solver
  • Plotter
