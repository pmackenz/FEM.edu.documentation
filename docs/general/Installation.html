Installation

FEM.edu is easy to install from PYPI using the standard python3 pip command [1].

$ pip install femedu

This will install the actual package in your site-packages folder.

You should be ready to go.

Updating to the latest version

FEM.edu is still under development and both new features and bug fixes happen frequently. You can update your copy to the latest version by executing the following command [1].

$ pip install --upgrade femedu

Requirements

  • numpy

  • scipy

  • matplotlib

  • pandas

Footnotes

[1] (1,2)

The $ sign stands for the common Unix command prompt and shall not be entered by the user. The same command workd on Windows systems.
